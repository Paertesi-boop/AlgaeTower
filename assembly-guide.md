# Assembly Guide / 組裝指南

## Step-by-Step Assembly / 步驟式組裝

1. **Base Assembly / 底座組裝**  
   Install the three-point support base, counterweight, and shock-absorbing pads. Ensure the base is level and stable.

2. **Layer 4 Installation / 第4層安裝**  
   Install the Stirling engine, battery pack, condenser, and heat recovery pipes.

3. **Layer 3 Installation / 第3層安裝**  
   Install the stirrer, temperature control module, extrusion head, and UV feeding box.

4. **Layer 2 Installation / 第2層安裝**  
   Install the spiral screw, wedge wire mesh, backwash nozzles, water ring pump, and ultrasonic module.

5. **Layer 1 Installation / 第1層安裝**  
   Install the compartment partitions, vertical LED tubes, high-reflectivity mirror walls, Venturi system, and UV feeding box.

6. **Piping & Electrical Connections / 管線與電氣連接**  
   Connect the water ring pump, gas lines, power cables, and control system.

7. **Testing & Initialization / 測試與初始化**  
   Fill with pure water and algae inoculum. Perform airtightness test and dry run.
